git init 初始化项目
git --version 查看git版本
git config --global user.name "Your Name"  用户名
git config --global user.email "email@example.com"  邮箱

git add .  添加到暂存盘
git checkout 文件名 从暂存盘还原文件

git comit -m "提示文本"  提交到版本库

git log 查看版本库历史

git reset --hard 哈希值  当前版本重新指向哪个版本

git reflog 历史版本

 git rest --hard HEAD^  回退到上一个版本 可以100^


git checkout -b dev  创建并切换到dev分支
git checkout master  切换的master分支
git branch mumu  创建一个mumu的分支
git branch  查看分支
git merge dev 把dev分支合并到 当前分支
git branch -d <name> 删除分支


添加远程分支
git remote add origin git@github.com/yzz3/testgit.git


上传
git push -u origin master
push 推送  -u 如果没有就创建  origin 远程  master 主要分支

git push -f origin master 强制本地替换远程

git push origin master 推送

git pull origin master 拉取主分支到本地

git clone git@github.com:jquery/jquery.git
复制网络jquery 文件到本地

在github 有个特殊的分支 gh-pages  可以做个人主页

github 项目的主页
本地
git checkout -b gh-pages
删除其他的内容
添加 index.html about.html...

git add .  添加到暂存盘
git commit -m "website" 提交到版本库
git push -u origin gh-pages  上传服务器
